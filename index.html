iPAS Quick Start for Project Managers and Their Designates (H2 Header)

TO START A NEW ACTIVITY SHEET: (H3 Header)

  • Go to http://gww.th.gov.bc.ca/ipas/
  • In left Nav bar, select "New Activity Sheet"
  • Begin filling fields, starting with your Project-Product number.
  • Save your input using yellow buttons at bottom of page
  • Mandatory fields are marked with a red asterisk (*). These must be completed before a project sheet can be submitted to the Moderator.
  • Remember: there are five steps to completing a PAS. The steps are listed in the left Nav bar:
    • General Info
    • Budget/Funding Information
    • Schedule
    • Traffic Management
    • Add Images (if available)
  • For detailed information regarding each of the fields see "iPAS Forms Fields Help"
